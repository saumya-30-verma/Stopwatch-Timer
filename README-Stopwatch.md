🕒 **React Stopwatch App :**

  A simple and responsive stopwatch application built using React and Vite.
  This app allows users to start, stop, and reset a timer with a clean and centered user interface.

✨ **Features:**

 - ▶️ Start the timer

 - ⏸ Stop the timer

 - 🔄 Reset the timer

 - ⚡ Uses useState and useRef hooks

 - 🎨 Clean and centered UI design

🛠️ **Tech Stack:**

 - React

 - Vite

 - JavaScript (ES6+)

 - CSS

📦 **Installation & Setup:**

 - Clone the repository:

       git clone https://github.com/saumya-30-verma/Stopwatch-Timer.git


 - Navigate into the project folder:

       cd Stopwatch-Timer


 - Install dependencies:

       npm install


 - Run development server:

       npm run dev

🔨 **Build for Production:**

     npm run build

🧠 **How It Works:**

 - `useState` is used to manage timer state.

 - `useRef` stores the interval ID.

 - `setInterval()` increments the time.

 - `clearInterval()` stops the timer.
